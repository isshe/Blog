# 凯撒密码
通过将明文中使用的字母表"平移"来进行加密。
秘钥空间：26
破解方式：直接暴力破解。

# 简单替换密码
* 更改26个字母的对应关系。如A表示X（A不再是A）。
* 秘钥空间：26 x 25 x 24 x ... x 1 = 403291461126605635584000000
* 破解方式：频率分析
